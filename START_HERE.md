## 1. Задача

На доске `n × n` выбираются точки одного шахматного цвета (`x+y ≡ 0` или `1 mod 2`). Нужно максимизировать их число при условии, что никакие три выбранные точки не лежат на одной прямой любого наклона. Максимум обозначается `D_mono(n)`.

Проверять нужно **все прямые**, а не только строки, столбцы и диагонали `±1`.

---

В `22x22/config_33.json` есть проверенная конфигурация из 33 точек:

```bash
python 22x22/verify_22x22.py
```

Она доказывает `D_mono(22) ≥ 33`. Нужно либо найти 34 точки, либо строго доказать их невозможность.

Для чётной доски две чётности равносильны отражением `(x,y) -> (21-x,y)`, поэтому достаточно одной чётности. Каждая точная модель обязана содержать ограничения для всех максимальных прямых, содержащих не менее трёх разрешённых точек.

## 6. Рациональный сертификат и главная диагональ

Для цели 34 используется сертификат:

```text
знаменатель: 187
целевая сумма весов: 6470/187 ≈ 34,598930
запас при 34 точках: 6470 - 34*187 = 112
```

Он вынуждает три диагонали содержать ровно по две точки:

```text
x-y = -2
x-y = 0
x-y = 2
```

Следовательно, на главной диагонали `x=y` обязательно выбрана ровно одна пара. Из 231 пар после поворота на 180 градусов остаётся полный набор из 121 канонического случая.

## 9. Правила тяжёлых запусков

1. Сначала прочитать этот файл.
2. Проверить очередь Actions, SHA и отсутствие дублирующего запуска.
3. Перед длинным запуском провести короткую проверку движка, разбиения, JSON, сборщика и загрузки артефактов.
4. Использовать закреплённую версию `ortools==9.15.6755` во всех заданиях, включая сборщик.
5. Seed приводить к знаковому 32-битному диапазону и сохранять исходное и фактическое значения.
6. Использовать до 20 параллельных заданий и 4 потока CP-SAT на задание.
7. Всегда сохранять артефакты через `if: always()`.
8. Отличать математический `UNKNOWN` от технической ошибки.
9. Не закрывать доску, пока остаётся хотя бы один `UNKNOWN`, пропуск или отменённый случай.
10. После завершения сохранять результаты в папке размера, обновлять README и этот файл.
